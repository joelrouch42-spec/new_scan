🎯 System Overview

A sophisticated Python-based trading system that analyzes NASDAQ 100 stocks using multiple technical indicators and executes automated trades through Interactive Brokers (IBKR).

Key Features

  • Multi-Indicator Analysis: Squeeze Momentum, MACD, SMC Order Blocks, Support/Resistance, ADX
  • Real-time & Backtest Modes: Live scanning with IBKR or historical backtesting with Yahoo Finance
  • Professional Risk Management: Exchange-managed bracket orders with automatic stop-losses
  • Interactive Charts: HTML charts with signal markers and technical overlays
  • Timezone Management: EST-aware scanning for US market hours
  • Asynchronous Architecture: Non-blocking IBKR connections preventing heartbeat failures
Market Focus: System is optimized for NASDAQ 100 stocks with US market hours (9:30-16:00 EST)

NEW Professional bracket order execution with exchange-managed stop-losses.

Features:

  • Parent market order for entry
  • Child stop-loss order linked via parentId
  • Automatic stop-loss calculation based on price action
  • 24/7 protection without script dependency
# Example bracket order structure: parent_order = MarketOrder('BUY', 100) parent_order.transmit = False stop_order = StopOrder('SELL', 100, stop_price) stop_order.parentId = parent_order.orderId stop_order.transmit = True
calculate_sl_level(symbol, action, high_price, low_price, previous_high, previous_low) → float

Calculates stop-loss levels based on price action and signal context.

Logic:

  • BUY positions: SL = min(signal_low, previous_low)
  • SELL positions: SL = max(signal_high, previous_high)
